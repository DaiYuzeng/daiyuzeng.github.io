关于

我是一名前端工程师，热衷于打造兼具可访问性、可扩展性和高性能的用户界面。我注重代码的清晰与可复用，喜欢在设计与工程的交汇处工作，把复杂需求转化为直观、易用的体验。

最近，我参与了 SUSE 核心产品 Harvester 的开发，负责设计可扩展的组件架构、管理复杂状态，并构建可复用的 UI 组件。我与 UI 团队紧密合作，优化交互流程，帮助实现“开箱即用”的用户体验。

此前，我在 Rancher 提升了产品的可访问性，确保符合 WCAG 2.1 标准，并通过 RESTful API 集成和 Playwright 端到端测试增强了产品的可靠性。更早在 东软集团，我开发了跨平台电动汽车租赁应用，并优化了 CRM 系统的审批流程。同时，我也持续为 Rancher 与 Harvester 等开源项目做出贡献，延续我让复杂技术更易用的承诺。

闲暇时我迷上了烘焙——我的一些“小作品”稍后会在网站上展示给大家。同时，我还是攀岩运动的初学者，所以我的业余生活常常伴随着烤箱与厨师机轰隆隆的声音，以及在攀岩馆“扑通”落地的声音 :D
